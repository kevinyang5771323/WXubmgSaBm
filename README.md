# 项目简介

## 核心功能
- 登录注册
- 图书信息管理
- 读者信息管理
- 图书借阅
- 图书归还
- 借阅记录查询
- 修改密码

## 适用场景
- 图书馆信息化管理
- 学校图书馆管理
- 企业图书资料管理

## 技术特点
- 使用Java语言开发，采用SpringBoot框架
- 数据库使用MySQL
- 前端界面采用Vue技术
- 提供详细的运行教程，方便快速部署和使用
